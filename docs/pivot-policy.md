# Politique de pivot

## Regle actuelle

Le pivot actif du projet est le francais (`fra`).

Les contributions prioritaires sont :

- `fra` <-> `lin`
- `fra` <-> `kon`
- `fra` <-> `swa`
- `fra` <-> `lua`

## Place de l'anglais

L'anglais (`eng`) fait partie de la vision du projet, mais il n'est pas encore le pivot principal.

Pour l'instant :

- on n'organise pas la collecte principale autour de `eng`
- on ne priorise pas `eng` <-> `fra`
- on ne priorise pas `eng` <-> langues congolaises

## Evolution possible

Une fois que les paires avec pivot francais seront :

- relues
- corrigees
- suffisamment valides
- structurellement coherentes

alors il sera possible :

- d'ajouter `eng` comme pivot secondaire
- d'ouvrir davantage de contributions `fra` <-> `eng`
- d'etendre ensuite vers `eng` <-> langues congolaises si cela devient utile

## Regle de contribution

Si un contributeur souhaite travailler avec l'anglais avant cette phase, cela doit rester exceptionnel, clairement identifie et ne pas prendre le dessus sur la priorite du pivot francais.
